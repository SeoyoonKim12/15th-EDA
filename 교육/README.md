## 같은 점수, 다른 과정: 과정 기반 문항·학습자 단위 분석
#### 참여자 : 김서윤, 윤동영, 박성하, 이세원, 이지원, 배소윤
#### EDA 프로젝트 자료 소개
> * Dataset 
>   * [EdNet](https://github.com/riiid/ednet)


<br>



## EDA 프로젝트 요약

1. 프로젝트 주제 및 목적

	EdNET 데이터를 활용하여 결과 중심 평가를 넘어 학습 과정 중심 평가를 구현하기 위해 혼란도 지표와 행동 기반 학습자 군집화를 수행하였다.

	연구문제1. 문항난이도로 설명되지 않는 과정난이도를 학습 로그 기반 ‘혼란도’로 정량화할 수 있는가?

	→ 새롭게 정의한 과정 기반 지표 ‘혼란도’와 IRT 기반 지표 ‘난이도’로 문항 유형을 클러스터링, 유형별 패턴 분석 후 시스템 개선 방안 분석

	연구문제2. 학습유형을 학습자의 능력과 더불어 학습과정 변수로 군집화할 수 있는가?

	→ 학습자의 능력과 학습 로그 데이터를 가공하여 만든 user feature를 기반으로 학습 유형 클러스터링, 학습 유형별 특징 분석 후, 학습 방법 피드백 방안 분석

2. 데이터 전처리

	학습자: 연구문제 1,2에 필요한 로그(KT1, KT2)가 모두 존재하는 “유료사용자”로 코호트 확정

	문항: 선발된 학습자 코호트의 KT1 제출 로그를 기준으로 IRT 분석에 사용할 ‘문항 집합’ 확정
            
 
3. 분석 방법 및 결과
   
	(1) 문항 단위 분석에서는 결과 난이도로 문항반응이론(IRT)의 문항 난이도를 활용하였으며, 과정 난이도로는 응답 시간과 응답 변경 횟수를 기반으로 로지스틱 회귀분석을 적용하여 ‘혼란도’를 정의하였다. 두 지표를 활용하여 문항을 4가지 (고난이도-고혼란도, 고난이도-저혼란도, 저난이도-고혼란도, 저난이도-저혼란도) 유형으로 분류하고 선지선택의 불확실성, 응답시간 분산, 문항태그의 개수, 토익 파트 등의 변수를 분석하여, 문항 특성을 탐색적으로 파악하였다.

	(2) 학습자 단위 분석은 학습 연속 일수, 선지 선택 취소율, 자료 재생률 등 행동변수와 더불어 IRT의 능력을 함께 분석하여 4개의 학습자 집단으로 유형화하였다.

	(3) 문항유형과 학습자 군집을 통합적으로 연결하여 분석한 결과, IRT 능력 수준은 비슷하지만, 행동 유형은 상이한 두 집단을 집중적으로 비교하였다.
		    
4. 결론
   
	문항 단위, 학습자 단위, 문항 유형과 학습자 군집 분석을 통해 학습 결과적 수치를 넘어서 과정 중심 평가를 기대해 볼 수 있었다.
    
5. 아쉬운 점
   
	혼란도 지표의 불안정성, response_change 변수의 희소성, 문항메타정보 부재 등
